# Icons for Music Presence

## Requirements

- Inkscape (preferred version is 1.3, must be on PATH as `inkscape`)
- ImageMagick (preferred version is 7.1, must be on PATH as `magick`)
- Make
- Git Bash on Windows  
  or a Linux bash shell (not tested)

## Build

```
make
```
